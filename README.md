# 电商客服智能Agent 🛍️🤖

基于通义千问的智能电商客服系统，具备多轮对话记忆和专业电商知识库。

## 🌟 核心特性

- **智能对话记忆**：记住对话上下文，提供连贯体验
- **专业电商知识**：内置商品、售后、物流等专业知识
- **智能信息提取**：自动识别订单号、商品名等关键信息
- **稳定API调用**：优化的错误处理和重试机制

## 🚀 快速开始

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 配置API密钥（编辑 .env 文件）
# DASHSCOPE_API_KEY=your_api_key_here

# 3. 运行客服系统
python main.py
```

## 🎯 技术亮点

- **智能分类**：自动识别用户意图（商品咨询、订单查询、售后服务等）
- **上下文理解**：基于对话历史提供个性化服务
- **实体提取**：准确识别订单号、商品信息等关键数据
- **优雅降级**：支持通义千问和OpenAI双API备份
